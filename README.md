# Learning Apps

A collection of interactive study tools and learning applications.

## Apps

| App | Description | Status |
|-----|-------------|--------|
| [CY 2550 Study Tool](./cy2550-study-tool) | Cybersecurity midterm study engine with practice questions, exam simulation, and integrated course notes | ✅ Live |

## Deployment

Each app lives in its own directory and can be deployed independently via [Vercel](https://vercel.com).

To deploy a specific app, set the **Root Directory** in Vercel project settings to the app's folder (e.g., `cy2550-study-tool`).
