# CY 2550 — Study Engine

Interactive midterm preparation tool for CY 2550: Foundations of Cybersecurity (Northeastern University, Spring 2025).

## Features

- **Practice Mode** — Randomized questions across 15 topic areas with instant rubric grading and model answers
- **Exam Simulation** — 15-question timed exam mirroring real test conditions
- **Integrated Notes** — Full searchable course study guide with sidebar navigation, category filtering, and expand/collapse
- **Grade Prediction** — Session-based predicted grade using content accuracy, reasoning quality, topic coverage, and speed metrics

## Deployment

This is a single-file static app. Deploy by pointing any static host at this directory.

**Vercel:** Set root directory to `cy2550-study-tool` — no build step needed.
